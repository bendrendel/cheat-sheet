DOM Tools

  • Edit: Edit node content, attributes, type directly in Elements pane
  • Reorder: Reorder nodes by dragging
  • State: Force active, hover, etc. using right-click menu on node
  • Hide: Press H to hide a node
  • Console: Esc to open console drawer
  • Current Node in Console: Use $0 in the console to reference currently selected node
  • Global Variable: Right click on node and choose store as global variable to easily reference in console
  • Copy JS Path: Right click on node and choose copy JS Path to get a document.querySelector() for the node copied to the clipboard
  • Break on: Right click on node and choose break on... to pause JS execution if something about the node changes

Device Mode

  • Device Mode: Roughly simulates behavior on mobile devices, toggle on with icon in DevTools in upper left
  • Screen size: Enter in screen size or drag sliders to resize
  • Media queries: Use the option menu in the upper right to show media query breakpoints and change width to trigger them
  • Device Type: Set the device type to either mobile or desktop viewports and whether it has a circle cursor that fires on touch events, or normal cursor that fires on click events
  • Viewport Mode: Choose responsive to change screen size manually, or pick a specific device from the drop down, can use rotate button to switch rotation, and can choose to show or hide the device frame from the options dropdown. Can also create custom devices with their own name, dimensions, etc.
  • Rulers: From options dropdown, can show rulers measuring pixels
  • Zoom: Can use zoom dropdown to zoom in or out
  • Throttling: From the throttle dropdown, can choose online for wifi (no throttling?), mid-tier mobile to simulate fast 3G and slow down your CPU 4 times slower than normal, or low-tier mobile to simulate slow 3G and slow down CPU 6 times slower than normal. Can also throttle the network and CPU independently from the performance panel under the settings dropdown.
  • Geolocation: Under the main Customize and Control Devtools dropdown in DevTools, select More Tools > Sensors, to pick specific GeoLocations, enter your own coordinates or select location unavailable
  • Orientation: Under the main Customize and Control Devtools dropdown in DevTools, select More Tools > Sensors, to pick specific orientations like portrait upside-down, device up, etc.
