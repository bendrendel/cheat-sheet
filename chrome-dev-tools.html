DOM Tools

  • Edit: Edit node content, attributes, type directly in Elements pane
  • Reorder: Reorder nodes by dragging
  • State: Force active, hover, etc. using right-click menu on node
  • Hide: Press H to hide a node
  • Console: Esc to open console drawer
  • Current Node in Console: Use $0 in the console to reference currently selected node
  • Global Variable: Right click on node and choose store as global variable to easily reference in console
  • Copy JS Path: Right click on node and choose copy JS Path to get a document.querySelector() for the node copied to the clipboard
  • Break on: Right click on node and choose break on... to pause JS execution if something about the node changes
