HTTP Requests

JSON: JavaScript Object Notation, format in which data is sent in response

AJAX: Asynchronous JavaScript and XML, enables requests to be made after initial page load, initially for XML data but now used for requests in many different formats

XHR API: XMLHttpRequest API, an object used to create and send requests in JS, also initially for XML data but now supports other formats. This is a somewhat outdated way to make requests, fetch() is a modern more efficient version of it

Query string: appended to the end of a url after a '?', consists of one or more parameters separated by '&', parameters consist of key-value pairs separated by '=', e.g. https://myapi.com/endpoint?key1=value1&key2=value2

JSON.stringify(myObject): converts a value like an object to a JSON string, need to do this if sending a request with data
