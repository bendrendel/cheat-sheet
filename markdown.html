Markdown is a text-formatting language that is widely used in software/technical documentation, many communication applications like skype, major websites like wordpress, write blogs posts for a website, to take notes, implemented in emails, designed for the web with PERL

It is designed to be lighter weight than HTML to be more accessible, but can't do nearly as much as HTML

Markdown processors or parsers are included in certain software that can read markdown, it converts markdown to HTML

File extension is .markdown or .md

        #Heading Level 1
        ##Heading Level 2
        ##Heading Level 3

        **Ordered List**
        1. *One asterick italicizes
        2. **Two asterick bolds
        3. ~Tilde for strikethrough

        **Unordered list**
        * Item 1
        * Item 2
        * Could also use + or - instead of *
