Color Theory

Primary colors: Red, Blue, Yellow

Secondary colors created by mixing primary colors: Green, Orange, Purple

Warm colors: Reds, oranges, yellows, browns, tans, evoke warmth, aggression, boldness

Cool colors: Blues, purples, greens, most grays, evoke calmness, soothing, winter, water

Neutral colors: Colors with low lightness and/or saturation

Tint/Shade: Tint means increasing the lightness of a given hue, shade means decreasing the lightness of a given hue

Contrast: hue is most contrasting on opposite side of color wheel, lightness contrast is highest with 100% difference

Color Schemes

MDN Color Picker Adobe Color Cloudflare Design

Monochromatic: Uses shades and tints of a single hue, evokes organized, harmonic feeling

Complementary: Uses two hues 180deg apart, common for logos, dramatic display of color, can clash if large areas of the two hues are adjacent

Analogous: Uses multiple hues close to each other, often one dominate hue, a second support hue, and third accent hue, pleasing, calming, low-contrast, good for backgrounds, not good to draw attention

Triadic: Uses three hues 120deg apart, evoke equality, vibrancy, security

Best Practices

Neon colors: Use sparingly as they can be hard on the eyes

Vibrating effect: Created by adjacent colors with complementary hues and high saturation, avoid

Use Contrast Selectively: Limit overall contrast on page and use it selectively to highlight important things like actions the user should take

Foreground/Background Contrast: Contract Checker ensure text or foreground content is sufficiently contrasting with background, but extreme conftrast can also be hard to read like pure black on white. WCAG recommends text is 4.5 times lighter or darker in luminance than the background, unless font size is at least 24 pixels then can be 3 times lighter or darker.

Background colors: Neutral allows you to easily add high contrast where needed, dark for light text or assets, medium difficult to create enough contrast with text or foreground content

Brand colors: Good brands have a brand color, and this should take up 60% of the color on the site

Accent colors: Use to draw the users attention, such as in calls-to-action

Buttons: Good to use accent color for buttons, should have different states like hover, disabled; hover typically a shade or tint of button color

Input field: Should have states like selected and disabled, can have success and failure, selected typically uses a border color or box shadow effect

Semantic Colors: Red for errors, etc. see below

Whitespace: empty areas between elements, be sure to use it to help focus content, but not so much to cause issues trying to navigate seamlessly through content

Color blindness: Accessible Color Palettes some users cannot distinguish red-green well, also blue-yellow where blues appear green, and total color blindness, pair color indicators with symbols to help, like an x symbol with a red error message

Err on the side of usability over aesthetics if they are in conflict

Setting color with CSS

Foreground color set with color property, background color set with background-color property

Opacity or Alpha can be set with opacity property or in hsla or rgba between 0 (transparent) and 1 (opaque), amount of background to blend with each pixel

Color keywords provide limited options, 147 colors, including transparent which is equivalent to rgba(0,0,0,0)

rgb or rgba takes three values defining intensity of red, green, blue, each between 0-255, or 0-100%, 256*256*256=16,777,216 colors; rgba takes fourth value for alph from 0 to 1

Hex is exactly the same color spectrum as rgb but defined with three consecutive hexadecimal numbers (16 digit number system 0-9 A-F) for hexadecimal numbers that repeat two characters, you can represent them with one

hsl or hsla is most intuitive and semantic, hue is 0-360 (degrees), saturation 0%-100%, lightness is 0%-100%, alpha (for HSLA only) is 0 to 1
