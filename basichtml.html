Basic structure of HTML

File extension is .html

HTML is composed of elements

Most elements are composed of an opening tag, content, and a closing tag containing a slash

Some elements only consist of an opening tag, known as a self-closing tag, may include or omit slash at end of tag

The tags contain the element name such as p

Attributes can be included in opening tags and consist of a name and possibly value, like id="my-element"

Elements can be nested, creating a hierarchical tree of child/parent/sibling relationships, parents often referred to as containers

Whitespace in HTML file is ignored, and should be used to make the file easier to read, 2 space indentation is W3C convention

HTML is permissive when it comes to errors, meaning it still works even if there are syntax errors because browsers have built-in rules to interpret bad syntax. Markup Validation Service is a site to check for errors in your HTML

Important structural elements

HTML documents should always start with the !DOCTYPE html document type declaration to ensure browser interpret it correctly

They always need the html element, so the browser knows where the content is to interpret as html

Usually the head tag is the first child element of html and contains metadata that is not directly displayed on scree

The head contains the title element, which is displayed on the browser or tab title bar

Only content inside the body tag can be displayed on the screen

The div (division) element used for dividing page into blocks or sections

Important text elements

Six heading elements: h1, h2, h3, h4, h5, h6

The p (paragraph) element used for block of plain text

The span element used to separate small pieces of content that is inline with other text

The em (emphasis) element generally used to separate text inline as italic

The strong element generally used to separate text inline as bold

The br (break) element used to create a line break and is self-closing

Other important elements

  • The ul (unordered list) element
  • Must be used with the li (list item) element as children
  • To create a bulleted list
  1. The ol (ordered list) element
  2. Again used with the li element as children
  3. To create a numbered list

The img (image) element used to display images and is self-closing, must have src attribute

Should usually have an alt attribute, which creates a tooltip if image doesn't load, is more accessible for screen-readers, and better SEO


The a (anchor) used to create hyperlinks, must have href attribute

href can use "./" for relative paths, or reference an element id to jump somewhere on the same page, # alone will go to top of page

Content of a element can be text, or an img, or nearly any other element, which will turn it into a link

Can have a target attribute, when set to _blank will open link in new tab (new window on older browsers)
