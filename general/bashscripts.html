Bash Scripting

Bash Script or shell script, file extension is .sh, executes within a Bash shell interpreter terminal, any command you can run in the terminal can be run in a Bash script, useful for frequently used sets fo commands

~/bin/ good practice to store your commonly used scripts in this directory, make sure this directory is in your PATH environment variable by including PATH=~/bin:$PATH in your bash profile

Executing a Script

  • chmod +x script.sh run this command in your terminal first to give the example script.sh file execute permission

  • ./script.sh now execute the script with this command


Bash Script Basics

#!/bin/bash <-- always include in first line of script, specifies what interpreter to use for the script

echo "Hello World" <-- will print Hello World in the terminal when you execute the script

greeting="Hello Again, World" <-- declare a variable by setting the name equal to a value, no spaces around equal sign

echo $greeting <-- access variables by prepending them with a dollar sign, put string variables in double quotes when comparing, e.g. "$foo" == "$bar"

numone=2

numtwo=3

numone=$((numone + numtwo)) <-- syntax for arithmetic with variables

echo "when you add numone to numtwo, you get $numone"<-- including a variable in a string


Bash Script Conditionals

index=3

if [ $index -lt 5 ]

then 

   echo $index

else 

   echo 5

fi

Number comparison operators:

  • -eq equal
  • -ne not equal
  • -le less than or equal
  • -lt less than
  • -ge greater than or equal
  • -gt greater than
  • -z is null

String comparison operators:

  • == equal
  • != not equal

Bash Script Loops

for loop: iterate through a list, e.g. $paragraph is list of words, word variable is declared in for statement


            for word in $paragraph
            do
                echo $word
            done

while loop: iterate while a condition is true, 


            while [ $index -lt 5 ]
            do
                echo $index
                index=$((index + 1))
            done

until loop: iterate until a condition is true


            until [ $index -eq 5 ]
            do
                echo $index
                index=$((index + 1))
            done

Bash Script Inputs

read: prompts for input and saves to a variable (e.g. number below)


            echo "Guess a number"
            read number
            echo "you guessed $number"

input arguments: user adds them when they run the script (see below), access them in the script with $1, $2, etc. where $1 is the first argument

./guessnumbers.sh 4 50

in your script if you need to accept an indefinite number of arguments, then you can iterate through them with the "$@" syntax


            for number in "$@"
            do
                echo "you guessed $number"
            done

external files: you can assign a whole set of files to a variable using regex (e.g. below all files in a directory) and loop through the files (e.g. below we print the full path and filename of each file)


            files=/some/directory/*
            for file in $files
            do
                echo $file
            done
