Example Website Style Guide

Website style guides should at least include colors, fonts, and text styles You could also document other components like button styles

Colors

Colors should specify a name (can name it yourself); a box displaying the color; the hex, rgb, rgba, hsl, or hsla value; and optionally a description of how it will be used

Text Styles

Text styles should specify the styling of different text elements to be used like headings and paragraphs

Main Page Heading (h1)

  • Font-family: Nunito Sans
  • Font-size: 26px
  • Font-weight: 700 (bold)

Subheading (h2)

  • Font-family: Poppins
  • Font-size: 18px
  • Font-weight: 500

Paragraph (p)

  • Font-family: Poppins
  • Font-size: 14px
  • Font-weight: 400 (regular)
