APIs

Basics

  • API stands for Application Programming Interface
  • Typically a layer of software on a server that handles operations on a database
  • Provides specific rules to outside applications for how to interact with the data, like how to retrieve data that the outside application can then use
  • Often data is returned as a JSON object, which you can manipulate to use in your application

API Services

  • Free APIs: Some APIs like Yelp offer their API to the public to expand their reach because they make money the more their data gets viewed
  • Paid APIs: Paid APIs like twilio offer valuable services to programmers for a monthly fee or per use fee
  • Internal APIs: Developers will use APIs in their apps to allow for easy expansion, so different versions of their software (web, mobile, etc.) can easily use the API to interact with the data

API Documentation

  • APIs will provide documentation of how to utilize the API
  • API Key this is often require to be registered with the API to use it, it should be kept secret because malicious users could use it to rack up charges on paid APIs
